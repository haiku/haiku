Bem-vindo ao Haiku!

Obrigado pelo seu interesse no Haiku! Este é o nosso quinto lançamento público. Esperamos atrair novos desenvolvedores para o projeto e oferecer aos utilizadores uma oportunidade de experimentar o Haiku. Apesar de esta versão beta incluir todas as funcionalidades principais planeadas para a versão final do Haiku, ainda tem muitos bugs e alguns traços que precisam de ser retocados. Consideramos, contudo, que esta versão é bastante estável e esperamos que lhe permita descobrir o grande potencial do Haiku e compartilhar do nosso entusiasmo.

Visite o site do nosso projeto em Haiku-os.org para saber das últimas notícias e comunicados.

Cuidado com os Bugs

Estivemos a trabalhar arduamente para corrigir bugs potencialmente críticos, entretanto, por ser uma versão beta, é natural que ainda existam alguns à espera de serem descobertos.

Desencorajamos o uso de versões beta como sistemas de produção. Os seus dados poderão ser seriamente alterados, corrompidos ou completamente destruídos. Mantenha as cópias de segurança atualizadas!

Uma das principais razões para lançar uma versão beta é para cativar mais pessoas para testar o sistema e descobrir falhas ou oferecer sugestões para futuros melhoramentos. Por favor relate erros e sugestões de melhoramentos no nosso sistema de acompanhamento de bugs, depois de usar a nossa ferramenta de pesquisa, para evitar duplicados.

  • Dicas para um relatório de erros útil

Conhecer o sistema

While the first release(s) of Haiku will be very much like the BeOS R5, the operating system it is reimplementing, there are many additions and improvements that were developed by the Haiku team over the years. So, if you're a returning BeOS user or are new user, have a look at these links:

  • Haiku Quick Tour
  • Guia do Utilizador do Haiku
  • Obter ajuda e suporte, e tornar-se parte da comunidade

Instalar mais software

A maior parte do software disponível pode ser descarregado e instalado com a aplicação HaikuDepot que vem com o Haiku.
Por causa da sua compatibilidade de código e binária, muitas aplicações do BeOS podem ser também executadas no Haiku. Muitas delas foram reempacotadas para o Haiku ou ainda correm sem modificações. Encontrará mais informações sobre como instalar novos programas no capítulo Aplicações do guia do utilizador.

Há várias coleções de software baseadas na comunidade. Consulte a página Software Sites no nosso website.

Atualizar o Haiku

A atualização do Haiku, bem como dos pacotes de terceiros, é feita com a aplicação SoftwareUpdater. Quando terminar, reinicie.
Para saber mais detalhes, consulte o guia Updating and downgrading your system (atualizar e reverter atualizações do seu sistema) no nosso website.

Wifi no Haiku

O Haiku suporta redes sem fio encriptadas com WPA/WPA2 ou WEP e, claro, abertas ou não encriptadas. Por motivos relacionados com as diferentes políticas de licenciamento dos diversos fabricantes de hardware, pode ser necessário um processo de instalação manual de firmware. Encontrará mais informação sobre o hardware suportado, instalação de firmware e como aceder a uma rede sem fios na secção Passo a passo: Redes sem fios do Guia do Utilizador do Haiku.

Conhecer a API

Uma vez que o Haiku é uma reimplementação, a API ainda é a mesma do BeOS R5 (com alguns acréscimos). Portanto, a documentação que consta do Be Book ainda é um recurso válido. Graças à ACCESS, que detém os direitos de antiga propriedade da Be Inc., temos permissão para fornecer uma cópia do be Book.
As divergências a partir da BeOS API e os acréscimos específicos do Haiku são reunidos no Haiku Book.

Há alguns recursos que lhe poderão ser úteis para ajudar a começar:

  • Comece a programar para Haiku
  • Explorador do Código-fonte
  • The Be Book
  • The Haiku Book
  • Be Newsletter Archive

Entrar em contacto

Entre em contacto com outros desenvolvedores e utilizadores, e junte-se às discussões no IRC (#haiku no web chat), nos nossos fóruns ou nas listas de discussão (mailing lists). Quando se juntam muitas pessoas diferentes, devem ser estabelecidas regras para manter efetiva a comunicação. Por favor, respeite a nossa Etiqueta nas Listas de Discussão.
