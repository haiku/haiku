¡Bienvenido a Haiku!

¡Gracias por poner Haiku a prueba! Este es nuestra quinta entrega al público. Esperamos atraer nuevos desarrolladores a este proyecto y darle a los usuarios una oportunidad para probar Haiku. Aún cuando esta primera versión beta incluye todas las principales características planeadas para la versión final de Haiku, todavía existen muchos fallos y algunas asperezas que hay que limar. No obstante, consideramos que esta entrega es muy funcional, y esperamos que le permita descubrir el gran potencial de Haiku, y que pueda compartir nuestro entusiasmo.

Visite el sitio web de nuestro proyecto Haiku-os.org para ver los últimos anuncios y noticias.

Cuidado con los fallos

Hemos trabajado muy duro para resolver todas los fallas más críticas, sin embargo, ya que es una versión beta, es muy probable que existan algunas por descubrir.

No recomendamos el uso de versiones beta en sistemas de producción. Sus datos podrían resultar alterados, estropeados o incluso destruidos. ¡Conserve copias de respaldo al día!

Una de las razones principales para liberar una versión beta es atraer más gente para poner a prueba el sistema y descubrir fallos o darnos sugerencias para futuras mejoras. Por favor, envíe informes de fallos y mejoras en nuestro bug tracker, luego de utilizar su sistema de búsqueda, para evitar duplicados.

Habituándose al sistema

Las primeras versiones de Haiku serán muy similares a BeOS R5, el sistema operativo que está recreando. Si nunca lo ha probado, puede mirar estos enlaces:

  • Guía de usuario de Haiku
  • Reciba ayuda y soporte y vuélvase parte de la comunidad

Instalar más software

La mayoría del software disponible se puede descargar e instalar con la aplicación HaikuDepot que viene con Haiku.
Debido a nuestra compatibilidad de código fuente y binario, muchas aplicaciones de BeOS funcionan bien en Haiku. Muchas de ellas han sido re-empaquetadas para Haiku o todavía corren sin necesidad de ser modificadas. Puede encontrar más información sobre cómo instalar nuevos programas en el capítulo Aplicaciones, en la Guía de Usuario.

Existen varias colecciones de software basados en la comunidad. Vea la página Sitios de Software (en inglés) en nuestro sitio.

Actualización de Haiku

Haiku y los paquetes de terceros se actualizan con la aplicacion SoftwareUpdater. Cuando la actualización esté completa, reinicie.
Para más detalles, vea la guía Actualizar su sistema en nuestro sitio.

Wifi en Haiku

Haiku soporta redes inalámbricas cifradas con WPA/WPA2 o WEP y claro también la redes abiertas, sin cifrado. Debido a diferentes políticas de licencia de varios fabricantes de hardware, es posible que requiera un proceso de instalación manual para el firmware. Una lista de dispositivos funcionando e instrucciones para instalación se pueden encontrar en el Taller: Redes inalámbricas de la Guía de Usuario Haiku.

Habituándose a la API

Como Haiku es una reimplementación, la API es la misma que para BeOS R5 (con algunos añadidos). Por lo tanto, la documentación existente en el BeBook es un recurso válido todavía. Gracias a ACCESS, quien tiene los derechos de propiedad de la antigua Be Inc., se nos permite distribuir una copia del BeBook. Las divergencias del API de BeOS y adiciones específicas a Haiku se recopilan en el libro Haiku.
