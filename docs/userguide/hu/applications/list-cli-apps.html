Parancssori programok listája

Útvonal:/boot/system/bin
/boot/common/bin
~/config/bin


A Haiku-val érkező parancssori programok a /boot/system/bin vagy a /boot/common/bin mappában találhatóak. Az általunk telepített programokat ajánlott a ~/config/bin mappában elhelyezni. Ezen útvonalak megtalálhatóak a PATH változóban is, így automatikusan megtalálhatjuk őket, nem kell hozzá tudnunk az útvonalat.
Ez a lista tartalmazza az összes, a Haiku-val érkező programot. Mindegyikhez egy rövid leírás is tartozik, azonban részletesebb segítséget is kaphatunk, ha a programot a --help paraméterrel indítjuk.
