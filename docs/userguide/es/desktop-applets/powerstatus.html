PowerStatus (Estado de energía)

Deskbar:Applets de Escritorio
Ubicación:/boot/system/apps/PowerStatus
Configuración:~/config/settings/PowerStatus settings

PowerStatus muestra información sobre el nivel de batería, así que sólo es útil en computadoras móviles. Si no se está ejecutando todavía, cuando inicie el applet le preguntará si debe abrir la ventana en modalidad ventana o si debe reposar en el Deskbar. En modalidad ventana podrá cambiar el tamaño del ícono si cambia el tamaño de la ventana y usando la manija del Replicante para arrastrarla al Escritorio.
Donde sea que esté funcionando, se opera desde el menú contextual que aparece con el botón derecho.

Nota: PowerStatus requiere soporte funcional de ACPI.

El menú contextual ofrece estas opciones:

Mostrar etiqueta de textoMuestra el nivel de batería restante en tiempo o porcentaje.
Mostrar ícono de estadoMuestra ícono del applet.
Mostrar tiempo / porcentajeAlterna entre mostrar el nivel de batería en porcentaje o tiempo restante (Mostrar etiqueta de texto debe estar activa).
Información de batería…Muestra la ventana de información ampliada de batería.
Acerca de…Muestra la ventana "Acerca de".
SalirCierra el applet PowerStatus.

Cuando Mostrar etiqueta de texto está activo, el nivel de batería se muestra entre corchetes mientras la carga está en curso.
