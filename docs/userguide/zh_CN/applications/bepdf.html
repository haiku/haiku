BePDF

桌面栏：应用程序
位置：/boot/apps/BePDF/BePDF
文档：/boot/apps/BePDF/docs/*
设置：/boot/common/settings/BePDF


BePDF 是一个快捷的PDF阅读器。除了用于阅读，对于未加密的PDF文档，它还支持添加注释和用户标签。目前它已经支持20种语言的本地化，并且可以简单的通过文本文件来添加其他语言支持。

有两种格式的文档资料： HTML 或者 PDF。后者可以通过 Help | Show Help...来打开。
